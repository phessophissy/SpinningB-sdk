# Contributing to Spinning Board SDK

Thanks for your interest in contributing!

## Development

```bash
git clone https://github.com/phessophissy/SpinningB-sdk.git
cd SpinningB-sdk
npm install
```

## Running Tests

```bash
npm test
```

## Pull Requests

1. Fork the repo and create your branch from `main`.
2. Add tests for any new functionality.
3. Ensure the test suite passes.
4. Submit your PR.
